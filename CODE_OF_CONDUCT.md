# Code of Conduct

## Our pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Our standards

**Examples of behavior that contributes to a positive environment:**
- Using welcoming and inclusive language
- Being respectful of differing viewpoints
- Gracefully accepting constructive criticism
- Focusing on what is best for the community

**Examples of unacceptable behavior:**
- Trolling, insulting/derogatory comments, and personal or political attacks
- Public or private harassment
- Publishing others’ private information without permission
- Requesting or contributing features designed to commit fraud or deceive employers

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and issues that do not align with this Code of Conduct.

Report issues via GitHub or the maintainer contact on the profile.
